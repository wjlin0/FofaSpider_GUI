#FofaSpider
FofaSpider 是基于Python3.7,pyside6,pyDracula,selenium无头浏览器 开发而成的爬虫GUI脚本。针对 fofa 提取关键信息

## 安装
```dos
git clone -m https://github.com/wjlin0/FofaSpider_GUI.git
```
## 功能
**1.支持csv导出**

**2.支持json格式导出**

**3.支持调整每个城市爬取的最大页数**

**4.fofa原生查询语句**

**5.支持导出最后一次查询历史记录**
